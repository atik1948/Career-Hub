# Career Hub

A React + Vite job portal demo where users can browse featured jobs, view details, and save applied jobs in localStorage.

## Tech Stack
- React 19
- React Router
- Tailwind CSS + DaisyUI
- Vite

## Routes
- `/` Home
- `/statistics` Statistics
- `/applied-jobs` Applied Jobs
- `/blog` Blog
- `/blog/:slug` Blog Details
- `/job/:id` Job Details

## Features
- Featured job listing
- Job details page
- Apply workflow using localStorage
- Applied jobs filter (All/Remote/Onsite)
- Responsive layout

## Run Locally
1. `npm install`
2. `npm run dev`
3. `npm run lint`
4. `npm run build`

## Data Source
Static JSON files:
- `data/jobs.json`
- `data/categories.json`
